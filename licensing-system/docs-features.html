✨ 功能指南

全面了解 OneClip 的强大功能

📋 剪贴板管理

🎯 自动记录历史

OneClip 自动保存您复制的所有内容，支持：

  • 文本：纯文本、富文本、代码
  • 图片：PNG、JPEG、GIF、WebP 等
  • 文件：所有文件类型
  • 视频：MP4、MOV 等
  • 音频：MP3、M4A 等
  • 文档：PDF、Word、Excel 等
  • 压缩包：ZIP、RAR 等

核心特性

  • 智能去重：自动识别并合并重复内容
  • 完整元数据：保留来源应用、时间戳、文件路径等
  • 容量管理：自动清理超过保留期限的内容
  • 收藏保护：收藏的内容永久保存

快速操作

  • 点击 - 复制到剪贴板
  • 空格 - 预览内容
  • 双击 - 编辑文本内容
  • 右键 - 显示完整菜单
  • 拖拽 - 拖拽到其他应用

🔍 智能搜索

⚡ 强大的搜索引擎

OneClip 提供多维度的智能搜索：

  • 内容搜索：搜索文本内容、文件名
  • 来源搜索：按复制来源应用筛选
  • 时间搜索：按时间范围查找
  • 类型筛选：按内容类型分类
  • 模糊匹配：智能容错，不需要精确输入

搜索技巧

  • 实时搜索：输入即搜索，无需按回车
  • 组合筛选：搜索 + 分类同时使用
  • 搜索历史：自动记住常用搜索词

🗂️ 分类系统

📁 智能分类

OneClip 提供系统分类和自定义分类两种方式：

系统分类（自动）

  • 📋 全部 - 所有历史记录
  • ⭐ 收藏 - 标记为收藏的项目
  • 📝 文本 - 纯文本和富文本
  • 🖼️ 图片 - 所有图片格式
  • 📁 文件 - 文件和文件夹
  • 🎬 视频 - 视频文件
  • 🎵 音频 - 音频文件
  • 📄 文档 - PDF、Office 等
  • 💻 代码 - 代码文件

自定义分类

创建您专属的分类规则：

  1. 打开设置 → 分类管理
  2. 点击"新建分类"
  3. 设置分类名称和图标
  4. 添加匹配规则（内容包含、来源应用、文件扩展名等）
  5. 保存后自动生效
💡 使用场景示例

• 创建"工作"分类，匹配特定应用（如 Slack、企业微信）
• 创建"代码"分类，匹配 .swift、.py 等扩展名
• 创建"重要"分类，手动添加重要内容

⚡ 快捷回复

💬 文本模板系统

预设常用文本，一键快速输入，提升沟通效率。

使用方法

  1. 按 ⌘ + Option + R 打开快捷回复窗口
  2. 选择或搜索需要的模板
  3. 点击或按回车即可复制到剪贴板
  4. 在任何应用中粘贴使用

管理快捷回复

  • 添加模板：快捷回复窗口点击"+" 按钮
  • 编辑模板：右键点击模板 → "编辑"
  • 删除模板：右键点击 → "删除"
  • 分类管理：创建分类整理模板
  • 导入导出：支持备份和分享模板

支持的内容类型

  • 📝 纯文本 - 常用回复、邮件模板
  • 🖼️ 图片 - 常用图片、表情包
  • 📁 文件 - 常用文档、附件

⭐ 收藏功能

💫 永久保存重要内容

将重要内容标记为收藏，确保不会被自动清理。

如何使用

  • 添加收藏：点击项目右侧的星标图标
  • 取消收藏：再次点击星标图标
  • 查看收藏：点击左侧"收藏"分类
  • 批量管理：使用批量操作功能

收藏特性

  • ✅ 永久保存，不受自动清理影响
  • ✅ 支持云同步（如启用）
  • ✅ 独立备份和导出
  • ✅ 快速访问和搜索

👁️ 内容预览

按 空格键 可以快速预览任何内容，支持：

  • 文本预览：Markdown 格式化显示
  • 图片预览：高清大图展示
  • 文件预览：Quick Look 系统预览
  • 代码预览：语法高亮显示
  • 视频预览：内联播放
💡 预览技巧

• 预览窗口支持左右箭头键快速切换
• 支持多文件预览（如选中多个文件）

✏️ 编辑功能

双击文本项目可以打开编辑器，支持：

  • 实时预览（Markdown 渲染）
  • 语法高亮（代码）
  • 查找替换
  • 撤销重做
  • 自动保存

📥 栈粘贴板

📚 按顺序管理和粘贴

栈粘贴板是一个特殊的剪贴板管理工具，可以按照先进后出（LIFO）的顺序管理多个内容，非常适合需要批量粘贴的场景。

使用场景

  • 批量填写表单时，依次粘贴多个字段
  • 整理文档时，按顺序粘贴多段内容
  • 编程时，依次粘贴多行代码
  • 数据录入时，快速粘贴多个数据项

操作方法

  • 添加到栈：按 Control + Shift + C 将当前剪贴板内容添加到栈顶
  • 依次粘贴：按 Control + Shift + V 从栈顶依次取出并粘贴内容
  • 查看栈内容：通过栈粘贴板窗口查看和管理所有内容
  • 清空栈：可以一键清空所有栈内容
💡 提示：

栈粘贴板采用"后进先出"的顺序，最后添加的内容会最先被粘贴。这种设计非常适合需要倒序粘贴的场景。

🎯 拖拽容器

📦 临时存储空间

拖拽容器是一个临时暂存区，可以收集多个项目后统一处理。

使用场景

  • 收集多个文件准备批量发送
  • 暂存稍后需要用到的内容
  • 整理素材和资料

操作方法

  • 打开容器：按 Control + Shift + D 呼出拖拽容器窗口
  • 添加内容：将文件/图片/文本拖拽到拖拽容器窗口
  • 使用内容：从容器中拖出项目到目标应用
  • 管理内容：可以删除单个项目或一键清空容器
💡 提示：

拖拽容器特别适合需要将多个文件或内容拖拽到同一个位置的场景，避免了反复切换窗口的麻烦。
