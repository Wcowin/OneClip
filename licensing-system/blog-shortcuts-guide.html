一、主界面快捷键

打开/关闭剪贴板历史

Cmd + Shift + V
最常用的快捷键。按下后会立即打开剪贴板历史面板，显示最近的复制记录。再按一次可以关闭。
💡 Pro Tip： 这个快捷键可以在设置中自定义，选择最适合你的组合。

快速搜索

打开历史面板后直接输入
打开历史面板后，无需点击搜索框，直接输入关键词即可搜索。支持文本、文件名、应用来源等多维度搜索。

收藏当前项目

Cmd + D
在历史面板中选中项目后，按 Cmd + D 可以快速收藏。收藏的项目会显示在收藏列表中，永不过期。

删除当前项目

Delete 或 Backspace
在历史面板中选中项目后，按 Delete 键可以删除该项目。删除后无法恢复，请谨慎操作。

二、快捷回复快捷键

打开快捷回复面板

Cmd + Shift + R
打开快捷回复面板，显示所有预设的快捷回复。可以快速搜索和插入常用文本。
💡 使用场景： 邮箱地址、电话号码、常用回复语、代码片段等。

快速搜索快捷回复

打开面板后直接输入
打开快捷回复面板后，直接输入关键词即可搜索。可以按名称、关键词、内容等多维度搜索。

三、粘贴板堆栈快捷键

打开粘贴板堆栈

Cmd + Shift + C
打开粘贴板堆栈面板。之后的每次复制操作都会自动加入堆栈，可以依次粘贴。
💡 使用场景： 填写表单时，先复制多个内容（姓名、电话、地址等），然后依次粘贴到对应字段。

粘贴堆栈中的下一项

Cmd + V（堆栈模式下）
在堆栈模式激活后，使用普通的 Cmd + V 会依次粘贴堆栈中的内容。每按一次会粘贴堆栈中的下一个内容，直到堆栈为空。

清空粘贴板堆栈

在堆栈面板中点击清空按钮
清空粘贴板堆栈中的所有内容。清空后无法恢复，请谨慎操作。

四、拖拽容器快捷键

打开拖拽容器

Cmd + Shift + D
打开拖拽容器窗口。可以拖拽内容到容器中暂存，方便整理和管理。
💡 使用场景： 设计师整理素材、程序员管理代码片段、写作者收集灵感。
⚠️ 注意： 拖拽容器功能默认关闭，需要在设置中启用。

从容器中快速粘贴

直接拖拽到目标位置
从拖拽容器中直接拖拽内容到目标应用或位置，实现快速粘贴。

六、快捷键自定义方法

如何自定义快捷键？

  1. 打开 OneClip 设置（Cmd + ,）
  2. 进入"快捷键"选项卡
  3. 找到要修改的快捷键
  4. 点击快捷键输入框
  5. 按下你想要的快捷键组合
  6. 点击"保存"按钮
💡 建议： 选择最顺手的快捷键组合，避免与系统快捷键冲突。

七、快捷键冲突解决

⚠️ 常见快捷键冲突：
  • Cmd + Shift + V 可能与其他应用冲突
  • Ctrl + Shift + C 可能与系统功能冲突
  • 如果快捷键不生效，检查是否被其他应用占用

解决方法： 在 OneClip 设置中修改快捷键为其他组合，或在冲突的应用中禁用该快捷键。

八、快捷键使用技巧

🎯 提升效率的 5 个技巧

  1. 记住最常用的 3 个快捷键
    • Cmd + Shift + V（打开历史）
    • Cmd + Shift + R（快捷回复）
    • Cmd + Shift + C（粘贴板堆栈）
  2. 自定义快捷键为最顺手的组合
    • 根据你的工作习惯调整
    • 避免与常用应用冲突
  3. 充分利用粘贴板堆栈
    • 批量复制，依次粘贴
    • 大幅提升填表效率
  4. 预设常用快捷回复
    • 邮箱、电话、地址等
    • 一键插入，省时省力
  5. 使用拖拽容器管理素材
    • 整理不同项目的内容
    • 快速访问常用素材

常见问题

Q: 快捷键不生效怎么办？

A: 检查以下几点：

  • 确认 OneClip 已启动
  • 检查快捷键是否被其他应用占用
  • 尝试重启 OneClip
  • 在设置中重新设置快捷键

Q: 可以修改快捷键吗？

A: 可以。在 OneClip 设置中的"快捷键"选项卡，可以自定义所有快捷键。
